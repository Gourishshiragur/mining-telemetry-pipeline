# Enterprise Mining Telemetry Lakehouse Pipeline

## 🔥 Overview

End-to-end production-grade data engineering pipeline built using Azure Data Factory and Databricks.

## 🏗 Architecture

ADF → Databricks → Bronze → Silver → Gold → BI

## ⚙ Features

* Batch + Streaming ingestion
* SCD Type 2 implementation
* Audit & Control framework
* Watermark tracking & replay
* CI/CD using Databricks Asset Bundles
* Monitoring & error handling

## 🚀 Tech Stack

* Azure Data Factory
* Azure Databricks (PySpark)
* Delta Lake
* Git + CI/CD

* Designed enterprise-level scalable pipeline
* Implemented data quality validation & deduplication
